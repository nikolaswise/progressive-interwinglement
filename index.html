Testing Web Components

How can we use a web component to progressively enhance an RDFa-enriched link with a preview element?

How Does it work?

  1. Create two HTML pages with RDFa data in their markup. Create a link between the two with the is="rdf-related" attribute on the a tag.
  2. In a js file, register a new HTML custom element that extends the a tag. Load it on the HTML page that has the link.
  3. Parse the resulting rdfa string to rudimentary json-ld.
  4. Adjust the DOM node to attach the fetched data to the UI.
